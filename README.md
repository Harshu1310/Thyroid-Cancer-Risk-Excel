# 🏥 Thyroid Cancer Risk Analysis Dashboard

## 🔍 Overview
An interactive Excel dashboard analyzing thyroid cancer risk factors 
to support healthcare-related insights and decision-making.

## 🛠️ Tool Used
Microsoft Excel (Advanced)

## 📁 Dataset Description
| File | Description |
|---|---|
| thyroid_cancer_risk_data.csv | Raw patient and risk factor data |
| dashboard.11123.xlsx | Interactive Excel dashboard file |

## 📌 Key KPIs Tracked
- Risk distribution by age group and gender
- Regional cancer prevalence
- Key risk factor frequency
- Diagnosis trend over time

## ⚙️ Process
1. Cleaned raw data using Power Query
2. Handled missing values and removed duplicates
3. Built Pivot Tables for multi-dimensional analysis
4. Designed interactive dashboard with slicers and charts

## 💡 Key Insights
- Higher risk identified in age group 40-60
- Identified top risk factors contributing to diagnosis
- Regional patterns found to support healthcare planning

## 🧠 Skills Used
Excel | Pivot Tables | Power Query | XLOOKUP | Charts | Slicers | Dashboards
